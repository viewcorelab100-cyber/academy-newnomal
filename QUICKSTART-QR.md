# 🚀 QR 코드 자동 출석 시스템 - 빠른 시작 가이드

## 📋 개요

학생이 **QR 코드**로 자동으로 출입을 체크할 수 있는 시스템입니다.

---

## 🛠️ 설치 및 실행

### 1단계: 데이터베이스 마이그레이션

```bash
# 방법 1: 배치 파일 실행 (권장)
migrate.bat

# 방법 2: 직접 실행
python scripts\migrate_add_qr_fields.py
```

**추가되는 컬럼:**
- `kakao_user_id` - 카카오 계정 ID
- `invite_token` - 초대 토큰
- `invite_expires_at` - 초대 만료 시간
- `qr_code` - QR 코드용 고유 ID

---

### 2단계: 서버 시작

```bash
start.bat
```

또는

```bash
uvicorn app.main:app --reload
```

서버가 http://localhost:8000 에서 실행됩니다.

---

## 🎯 사용 방법

### [관리자] 학생 초대

1. **학생 목록 페이지**로 이동
   ```
   http://localhost:8000/students-list
   ```

2. 학생 옆에 있는 **"📱 초대하기"** 링크 클릭

3. 초대 링크가 자동으로 생성됩니다
   ```
   http://localhost:8000/student/signup?token=abc123...
   ```

4. **"📋 링크 복사하기"** 버튼 클릭

5. 카카오톡으로 학부모에게 전송

6. QR 코드도 함께 표시됩니다 (학생이 사용)

---

### [학생] 가입하기

1. 관리자가 보낸 **초대 링크** 클릭

2. **"💬 카카오 로그인으로 3초 가입"** 버튼 클릭
   - MVP에서는 자동으로 시뮬레이션됩니다
   - 프로덕션에서는 실제 카카오 로그인 사용

3. 가입 완료 후 **QR 코드 페이지**로 자동 이동

4. QR 코드를 **스크린샷**으로 저장하거나 즐겨찾기

---

### [학생] 출입 체크

#### 방법 1: QR 스캔 (자동)

1. 학원 도착

2. 스마트폰에서 **QR 코드 표시**

3. 입구 태블릿/단말기에서 **QR 스캔**

4. **"등원" 또는 "하원"** 버튼 클릭

5. ✅ 자동으로 출석 체크!

6. 학부모에게 **카카오톡 알림** 자동 발송

#### 방법 2: 테스트 (수동)

1. QR 코드 페이지에서 **"테스트: QR 스캔 시뮬레이션"** 버튼 클릭

2. 등원/하원 선택

3. 출석 체크 완료

| 페이지 | URL | 설명 |
|--------|-----|------|
| 학생 목록 | `/students-list` | 관리자용 |

## 🎬 전체 시나리오

### 시나리오 1: 신규 학생 초대

```
1. 관리자 로그인
   ↓
2. 학생 목록 → "📱 초대하기" 클릭
   ↓
3. 초대 링크 복사
   ↓
4. 카카오톡으로 학부모에게 전송
   ↓
5. 학생이 링크 클릭 → 카카오 로그인
   ↓
6. 자동으로 연동 완료
   ↓
7. QR 코드 표시 → 스크린샷 저장
```

### 시나리오 2: 출입 체크

```
1. 학생이 학원 도착 (09:00)
   ↓
2. QR 코드 표시
   ↓
3. 입구 태블릿에서 QR 스캔
   ↓
4. "등원" 버튼 클릭
   ↓
5. ✅ 출석 체크 완료!
   ↓
6. 학부모에게 카카오톡 발송:
   "김철수 학생이 09:00에 등원했습니다"
```

## 🔧 API 테스트

### 1. 초대 링크 생성

```bash
curl -X POST "http://localhost:8000/students/1/invite" \
  -H "Authorization: Bearer {your_token}"
```

### 2. 학생 가입

```bash
curl -X POST "http://localhost:8000/student/signup" \
  -H "Content-Type: application/json" \
  -d '{
    "invite_token": "abc123...",
    "kakao_user_id": "kakao_test123"
  }'
```

### 3. QR 체크인

```bash
curl -X POST "http://localhost:8000/student/checkin?qr_code=qr_abc123&type=in"
```

## 💡 자주 묻는 질문

### Q1. 초대 링크가 만료되면?

**A:** 관리자가 다시 "📱 초대하기"를 클릭하여 새 링크를 생성합니다.

### Q2. 학생이 QR 코드를 잃어버리면?

**A:** QR 코드 URL로 다시 접속하거나, 관리자에게 요청하여 재발급받습니다.

### Q3. 하루에 2번 등원 체크가 필요하면?

**A:** 관리자가 수동으로 출석 수정 (`PATCH /attendance/:id`) 가능합니다.

### Q4. 실제 카카오 로그인은 어떻게?

**A:** 프로덕션에서는 카카오 개발자 센터에서 앱 등록 후 JavaScript SDK를 사용합니다.
